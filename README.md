# InfoGui
A clean, lightweight in-game HUD for A Township Tale that keeps track of system telemetry and your installed mods.

### Features
* **Telemetry stats**: Shows smoothed FPS, PC uptime, online player count, and your current chunk coordinates.
* **Mods directory**: Detects and displays all MelonLoader `.dll` files installed in your `Mods` folder, wrapped in a collapsible menu.
* **Log spam fix**: Silences the annoying Oculus legacy touchpad errors (`UnityEngine.Input` system exceptions) that constantly flood the console logs.
* **Futuristic HUD**: Custom glassmorphism layout with clean styling, thin borders, and neon cyan accents.

### How to use
* **I** key: Show/hide the GUI panel.
* Click and drag the top title area (`Game info`) to move the window.

### Installation
1. Make sure you have MelonLoader set up.
2. Download `InfoGui.dll` from the Releases tab on GitHub.
3. Put the DLL into your game's `Mods` folder.
